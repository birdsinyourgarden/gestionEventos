1. Evento click
Crea un botón que muestre un mensaje por consola cuando el usuario haga clic sobre él.

2. Cambiar texto

Crea un párrafo y un botón. Al hacer clic en el botón, el texto del párrafo debe cambiar a:

“Texto modificado con JavaScript”

3. Contador de clics

Crea un botón que cuente cuántas veces ha sido pulsado y muestre el número en un <span>.

4. Evento mouseover

Crea un div que cambie de color cuando el ratón pase por encima y vuelva a su color original cuando salga.

5. Mostrar texto en tiempo real

Crea un campo de texto y muestra en un párrafo el contenido mientras el usuario escribe.

6. Contador de caracteres

Crea un input de texto y muestra la cantidad de caracteres escritos.

7. Capturar datos del formulario

Crea un formulario con un campo nombre y un botón Enviar.
Al enviar el formulario:

Evita que la página se recargue.

Muestra el nombre por consola.

8. Validar campo obligatorio

Crea un formulario con un campo email.
Si el campo está vacío:

Evita el envío.

Muestra un mensaje de error.

9. Validación de contraseña
Crea un formulario con un campo contraseña.
La contraseña debe tener al menos 8 caracteres.

10. Validar dos campos
Crea un formulario con los campos usuario y contraseña.
Ambos deben estar completos para permitir el envío.

11. Select dinámico
Crea un <select> con tres opciones.
Muestra en un párrafo la opción seleccionada cada vez que cambie.

12. Confirmación de envío

Antes de enviar un formulario, muestra un cuadro de confirmación (confirm()).

13. Checkbox y botón

Crea un formulario con un checkbox “Acepto los términos”.
El botón Enviar solo debe activarse cuando el checkbox esté marcado.

14. Validación completa

Crea un formulario con:

Nombre: obligatorio

Email: formato válido

Edad: mayor o igual a 18

Muestra mensajes de error debajo de cada campo.

15. Validación en tiempo real

Crea un formulario que valide los campos en tiempo real usando el evento input.

Campos válidos → borde verde

Campos inválidos → borde rojo

16. Contador de envíos

Cuenta cuántas veces se ha intentado enviar un formulario, incluso si la validación falla.
