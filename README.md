1. Evento click
Crea un botón que muestre un mensaje por consola cuando el usuario haga clic sobre él.

2. Cambiar texto

Crea un párrafo y un botón. Al hacer clic en el botón, el texto del párrafo debe cambiar a:

“Texto modificado con JavaScript”

3. Contador de clics

Crea un botón que cuente cuántas veces ha sido pulsado y muestre el número en un <span>.

4. Evento mouseover

Crea un div que cambie de color cuando el ratón pase por encima y vuelva a su color original cuando salga.

5. Mostrar texto en tiempo real

Crea un campo de texto y muestra en un párrafo el contenido mientras el usuario escribe.

6. Contador de caracteres

Crea un input de texto y muestra la cantidad de caracteres escritos.

7. Capturar datos del formulario

Crea un formulario con un campo nombre y un botón Enviar.
Al enviar el formulario:

Evita que la página se recargue.

Muestra el nombre por consola.

8. Validar campo obligatorio

Crea un formulario con un campo email.
Si el campo está vacío:

Evita el envío.

Muestra un mensaje de error.
